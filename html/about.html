About

Welcome to Message in a Bottle, a creative web app designed to bring people together through short, meaningful messages. Inspired by the classic idea of sending notes across the sea, this platform allows users to write and share their own bottled messages for others to discover. Every message submitted becomes part of a rotating collection that appears throughout the day, making each visit unique. Whether it is a kind thought, a motivational quote, a funny line, or a personal reflection, every bottle carries something special. The app was built using HTML, CSS, JavaScript, Python, and Flask, combining front-end design with back-end functionality. JavaScript adds smooth animations and live countdown timers, while Python manages message storage, time-based message displays, and user interaction. Message in a Bottle was created to show how coding can be used to build something fun, interactive, and engaging. It is a digital space where words can drift from one person to another, just like a real bottle floating across the ocean. So take a look around, send a message, and see what arrives next.
